# WebTrack

## Explanation
WebTrack is a web application that allows you to observe what each user is doing on your website in real time.

The purpose of the app is to demonstrate how easy it is to be monitored online - often without even knowing it.

## Main functionalities
There is a main website that contains a control panel.

Inside that panel, you can see all of the different websites that are being monitored.

## Technologies
- Main programming language – C#
- Main framework – ASP.NET
- Front-end – Vue with TailWindCSS and ShadCN (Not using Next.js because Vue is lightweight and it integrates well with ASP.NET)
- WebSockets – SignalR for the real-time connection
